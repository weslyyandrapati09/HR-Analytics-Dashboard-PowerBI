# HR-Analytics-Dashboard-PowerBI
Interactive HR Analytics Dashboard built using Power BI, Excel, DAX and Power Query

HR Analytics Dashboard | Power BI

Overview

This project focuses on analyzing employee attrition and workforce trends using Power BI. The dashboard helps identify key factors influencing attrition across departments, salary groups, age categories, education fields, and employee demographics.

Tools Used

* Power BI
* Power Query
* DAX
* Excel / CSV

Business Problem

Employee attrition can impact productivity, hiring costs, and overall organizational performance. The objective of this project was to analyze HR data and uncover patterns that contribute to employee turnover.

Key Insights

* Analyzed data for 1,480 employees.
* Identified 238 employee attritions with an overall attrition rate of 16.08%.
* Sales department recorded the highest attrition rate at 20.67%.
* Employees aged 18–24 showed the highest attrition rate at 39.18%.
* Employees earning up to 5K experienced the highest attrition rate at 21.65%.
* Overtime employees showed significantly higher attrition compared to non-overtime employees.
